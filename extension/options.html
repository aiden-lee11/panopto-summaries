Panopto Summarizer Settings

Bring your own OpenAI or Gemini API key. Keys are stored only in this browser profile via chrome.storage.local.

How data is used

  • Captions are read from the active Panopto lecture page when you click summarize or copy captions.
  • Your selected transcript text is sent directly to the AI provider you choose.
  • API keys stay in local extension storage on this device unless you remove them.
  • Saved summary history stays local and can be cleared at any time from this page.

Before publishing

Link your privacy policy from the Chrome Web Store listing and describe the BYOK data flow clearly in the listing disclosure form.
