## Privacy Policy

`Panopto Summarizer` is a browser extension that helps users extract captions from Panopto lecture pages and send that text to an AI provider selected by the user for summarization.

### Data the extension accesses

- The extension reads caption or transcript text from the active `panopto.com` lecture page only when the user clicks an action such as summarize or copy captions.
- The extension stores user preferences locally in the browser, including prompt settings, selected provider, saved summary history, and any API keys the user chooses to enter.

### How data is used

- Transcript text is used only to fulfill the user-requested action.
- When the user runs a summary, the selected transcript text is sent directly to the provider chosen by the user:
- `OpenAI`
- `Google Gemini`
- API keys entered by the user are stored locally in `chrome.storage.local` so the extension can make requests on the user's behalf.

### Data sharing

- The extension does not sell data.
- The extension does not send transcript text to any server controlled by the extension developer.
- Transcript text is shared only with the AI provider selected by the user in order to generate the requested summary.

### Data retention and control

- Stored settings remain in the local browser profile until the user clears them, removes the extension, or clears browser extension storage.
- Saved summaries remain in local extension storage until cleared by the user.
- Users can remove saved API keys and local summary history from the extension settings page.
